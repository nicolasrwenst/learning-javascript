# Estruturas de Controle

`if`, `else if`, `else`:
- Avaliam condições booleanas e executam blocos distintos.
- Use parênteses para condição e chaves para blocos.

`switch`:
- Útil para múltiplas comparações de mesma expressão.
- Use `break` para evitar falls-through (a menos que desejado).

Exemplo prático (veja `example.js`) mostrando `if/else` e `switch` com casos.
