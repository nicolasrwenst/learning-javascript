# Operadores em JavaScript

Aritméticos:
- `+` soma (também concatena strings), `-`, `*`, `/`, `%` (resto), `**` (potência)

Comparação:
- `==` igualdade com coerção de tipo
- `===` igualdade estrita (sem coerção) — preferir
- `!=`, `!==` desigualdade
- `>`, `<`, `>=`, `<=`

Lógicos:
- `&&` (AND), `||` (OR), `!` (NOT)
- Curto-circuito: `a && b` retorna `a` se falsy, senão `b`; `a || b` retorna `a` se truthy, senão `b`.

Exemplo prático (veja `example.js`) mostrando comportamento e diferenças entre `==` e `===`.
