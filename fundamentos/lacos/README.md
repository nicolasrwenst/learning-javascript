# Laços de Repetição

`for`:
- Estrutura clássica: `for (inicialização; condição; pós) {}`
- Boa para iterações com índice.

`while`:
- Executa enquanto a condição for verdadeira. Checa condição antes do bloco.

`do...while`:
- Executa o bloco ao menos uma vez, depois checa a condição.

Exemplo prático (veja `example.js`) mostrando `for`, `while` e `do...while`.
