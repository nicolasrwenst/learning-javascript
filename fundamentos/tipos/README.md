# Tipos de Dados em JavaScript

Principais tipos primitivos:
- `string`: texto. Ex: "olá".
- `number`: números (inteiros e floats). Ex: 42, 3.14.
- `boolean`: `true` ou `false`.
- `null`: valor intencionalmente vazio (tipo especial).
- `undefined`: variável declarada sem valor.

Outros tipos:
- `object`: objetos e arrays.
- `function`: funções são objetos chamáveis.

Operador `typeof` mostra o tipo (com algumas idiossincrasias: `typeof null === 'object'`).
